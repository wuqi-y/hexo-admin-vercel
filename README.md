# Hexo-Theme-Acrylic-Server

## 分支说明
本地部署请使用：deploy-localhost；  
vercel部署请使用：deploy-vercel；  
trunk目录是部署至vercel的主要开发分支，从trunk开发，分commit到各个对应的分支中；  
release-v1.0.0分支是发布版，现在还未发布，留意release即可，发布后，在这个分支会留一份备份分支；
